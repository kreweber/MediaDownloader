## 🚀 Установка

### Шаг 1 — Установи Python 3.12.7

Открой **Win+R**, введи `cmd` и нажми Enter.

В открывшемся окне выполни:

```bash
winget install Python.Python.3.12
```

> Или скачай вручную: [python.org/downloads](https://www.python.org/downloads/release/python-3127/)  
> ⚠️ При установке обязательно поставь галочку **"Add Python to PATH"**

---

### Шаг 2 — Обнови pip

В том же окне cmd:

```bash
python -m pip install --upgrade pip
```

---

### Шаг 3 — Перейди в папку проекта

**Закрой предыдущее окно cmd.**

Открой папку с проектом в Проводнике, кликни на адресную строку вверху, введи `cmd` и нажми Enter — откроется терминал уже в нужной папке.

---

### Шаг 4 — Установи зависимости

```bash
pip install PyQt6 yt-dlp
```

---

### Шаг 5 — Запусти

```bash
python main.py
```
